## 78. Subsets (Medium)
https://leetcode.com/problems/subsets/description/

Given an integer array nums of unique elements, return all possible subsets (the power set).

The solution set must not contain duplicate subsets. Return the solution in any order.

 

Example 1:
    
    Input: nums = [1,2,3]
    Output: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]
    Example 2:
    
    Input: nums = [0]
    Output: [[],[0]]
 

Constraints:
    
    1 <= nums.length <= 10
    -10 <= nums[i] <= 10
    All the numbers of nums are unique.
  
  😌java

    
    class Solution {
        List<List<Integer>> res = new ArrayList<List<Integer>>();
        public List<List<Integer>> subsets(int[] nums) {
            dfs(0, nums, new ArrayList<Integer>());
            return res;
        }
        public void dfs(int index, int[] nums, ArrayList<Integer> curList){
            res.add(new ArrayList<Integer>(curList));//record answer
    
            for(int i = index; i < nums.length; i++){//choose one option at current level
                curList.add(nums[i]);//include nums[i]
                dfs(i+1, nums, curList);//continue build deeper
                curList.remove(curList.size()-1);//undo choice
            }
        }
    }
