## Group Anagrams - Medium
https://leetcode.com/problems/group-anagrams/

Given an array of strings strs, group the anagrams together. You can return the answer in any order.

An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

 

Example 1:

    Input: strs = ["eat","tea","tan","ate","nat","bat"]
    Output: [["bat"],["nat","tan"],["ate","eat","tea"]]
Example 2:

    Input: strs = [""]
    Output: [[""]]
Example 3:

    Input: strs = ["a"]
    Output: [["a"]]


Constraints:

    1 <= strs.length <= 104
    0 <= strs[i].length <= 100
    strs[i] consists of lowercase English letters.
    
    
:cupid:java
    
    class Solution {
        public List<List<String>> groupAnagrams(String[] strs) {
            List<List<String>> result = new ArrayList<>();
            Map<String, List<String>> map = new HashMap<>();
    
            // "atn" -> ["nat , tan"] 
            for(String str:strs){
                //get the key
                char[] charArr = str.toCharArray();
                Arrays.sort(charArr);
                // String sortedKey = charArr.toString(); NOTE this is wrong because charArr.toString() does not turn the char array into "aet" or "ant". it gives something like [C@1b28cdfa
                String sortedKey = new String(charArr);
                if(!map.containsKey(sortedKey)){
                    List<String> list = new ArrayList<>();
                    list.add(str);
                    map.put(sortedKey, list);
                }else{
                    List<String> list = map.get(sortedKey);
                    list.add(str);
                }
            }
            for(List<String> list : map.values()) result.add(list);
            return result;
        }
    }

    
    Time: O(n · k log k)
    Space: O(n · k)
    
    Where:
    
    n = number of strings
    k = maximum string length
